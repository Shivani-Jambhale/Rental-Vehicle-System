# Vehicle Rental & Management System

A Java-based console application built in Eclipse to automate the vehicle rental process, from booking to returns.

## Core Features
* **OOP Implementation:** Built using Object-Oriented Programming principles to manage vehicle and customer data.
* **Streamlined Workflow:** Automates the rental process and handles transaction records.
* **Java Logic:** Developed as a robust backend tool within the Eclipse IDE.

## How to Run
1. Open the project in **Eclipse**.
2. Run the main Java file to start the console application.
